# `@bond/autoresearch`

Bounded self-improvement loop for coding-agent experiments.

- `src/runner.ts`: main autoresearch loop, experiment execution, git integration, and metric comparison.
- `src/openai-researcher.ts`: OpenAI-backed web research helper.
- `src/types.ts`: manifest schemas, runtime types, and package interfaces.
- `src/index.ts`: package exports.

Tests:

- `test/autoresearch-runner.test.ts`: manifest parsing, experiment control flow, and web research integration.
